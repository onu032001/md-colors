# md-colors

Material design colors in GitHub. Colors are picked by `--md-\[color name\]-\[color index\]`, like `--md-blue-6`.
[Demo](https://onu032001.github.io/md-colors/demo.html)

## Usage

Just put this macro if you want to use it in CSS.
```html
<link rel="stylesheet" href="https://onu032001.github.io/md-colors/colors.css">
```

Or Javascript is open-source:
```html
<script src="https://onu032001.github.io/md-colors/colors.js"></script>
```
